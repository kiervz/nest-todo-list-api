## Description

Sample implementation of CQRS Pattern using [NestJS](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migration
```bash
# generate migrations
$ npm run migration:generate -- db/migrations/NewMigration

# run migrations
$ npm run migration:run

# revert migrations
$ npm run migration:revert
```
